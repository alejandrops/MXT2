# Comparativas · Dirección

Comparación side-by-side de KPIs entre grupos, períodos o tipos de vehículo.

## Para qué sirve

Responder preguntas tipo "¿el grupo Norte mejoró más que el Sur?" o "¿este mes vs el anterior?" con visualización clara de las diferencias.

## Qué muestra

Selección de 2 o 3 entidades a comparar:

- **Panel de KPIs side-by-side** · cada entidad en su columna
- **Delta absoluto y porcentual** · cuánto cambió cada métrica
- **Sparklines temporales** · tendencia individual de cada una
- **Texto narrativo** · resumen automático de los hallazgos clave

## Modos de comparación

- **Grupos** · mismo período, distintos grupos
- **Períodos** · mismo grupo, distintos períodos (mes vs mes)
- **Tipos de vehículo** · mismo período, distintos tipos

## Cómo se usa

1. Elegir el modo arriba (grupos / períodos / tipos)
2. Seleccionar las entidades a comparar (max 3)
3. Período en el navegador
4. La comparativa se actualiza · texto narrativo arriba

## Tips

- Para reuniones ejecutivas · exportar a PDF (botón arriba derecha)
- Comparar más de 3 entidades hace difícil de leer · usar Comparativa de objetos
- Las **diferencias estadísticamente significativas** se marcan con `▲`/`▼`

## Permisos

CA y MA. OP normalmente no necesita esta vista.
